# Escape Game FTel

Welcome to the **Escape Game FTel** project! 

This project is built upon a forked starter kit. To keep the development process organized and easy to track, this repository is divided into two main branches:

**[`main` branch](https://github.com/HungVulkan/Escape_Game_FTel/tree/main):** This is the active development branch. All the game building, new features, logic implementations, and updates will be committed here.

**[`original` branch](https://github.com/HungVulkan/Escape_Game_FTel/tree/original):** This branch preserves the untouched, unmodified kit from when it was first forked. It serves as a baseline reference to see how much the project has evolved from its starting point.
